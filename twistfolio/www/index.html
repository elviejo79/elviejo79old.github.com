JagAcademy la escuela para desarrolladores de software ágiles.

1

Qué es?

JagAcademy es un nuevo programa para desarrolladores de software. Donde la práctica, el mentoreo, el trabajo en equipo y el trabajo real se anteponen a la teoría.

2

Quiénes están invitados?

El programa está diseñado para jóvenes recién egresados de la carrera de sistemas o que tengan menos de un año de experiencia laboral que quieren pulir sus habilidades.

3

Qué puedes esperar?

Al finalizar el programa estarás capacitado para desempeñarte como un desarrollador de software con las habilidades prácticas que desarrolladores usan todos los días.
